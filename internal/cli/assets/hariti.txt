Usage:
  hariti [global options] <subcommand> [subcommand options]

Global Options:
  -c, --config <file>       Path to bundles.hariti configuration file
      --config-dir <dir>    Path to configuration directory
      --data-dir <dir>      Path to data directory
  -d, --directory <dir>     (Deprecated) Alias for --config-dir
  -v, --verbose             Enable verbose output
  -h, --help                Show this help

Subcommands:
  install                   Synchronize repositories and deploy the active generation
  sync                      Synchronize repositories and lock revisions
  deploy                    Deploy the active generation
  dump-graph                Dump the resolved graph as JSON
